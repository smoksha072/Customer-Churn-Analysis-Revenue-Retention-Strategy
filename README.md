# Customer-Churn-Analysis-Revenue-Retention-Strategy

This project focuses on analyzing customer churn behavior and identifying actionable strategies to reduce churn and protect revenue. The analysis is built using Python for data preparation and Power BI for visualization.

-- Objective

To understand:

1. Why customers churn
2. Which customers are at high risk
3. How businesses can reduce churn using data-driven strategies

--  Key Insights
1. High churn (26.5%) indicates a significant retention problem
2. Customers on month-to-month contracts churn the most
3. New customers (0–12 months) are more likely to leave
4. Higher monthly charges are linked to increased churn
5. Electronic check users show higher churn compared to automatic payments

--  Business Recommendations
1. Encourage long-term contracts to improve retention
2. Focus on early customer engagement during the first year
3. Review pricing strategy for high-paying customers
4. Promote automatic payment methods for better retention
5. Apply targeted strategies:
6. Discounts for high-risk customers
7. Engagement campaigns for medium-risk
8. Upselling for low-risk

-- Conclusion

This project demonstrates how data can be used to move beyond reporting and support actionable decision-making, helping businesses reduce churn and improve customer lifetime value.
